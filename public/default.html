A diaspora pod with advanced privacy policies that give you more control over your information.

Pod temporarily setup for testing secure photo sharing using Attribute Based Encryption

Instructions:

In order to test our solution we have created 3 users: Alice, Bob and Charlie. The system has an attribute universe consisting of the following attributes: friend(Alice),friend(Bob),friend(Charlie). Each user has the following attributes:

    Alice: friend(Alice),friend(Bob),friend(Charlie); Bob: friend(Alice),friend(Bob); Charlie: friend(Bob),friend(Charlie).

You can login with the usernames alice, bob and charlie and password 123456. To login click on the botton in top right corner of the page. To encrypt a picture, first click twice on a photo from the stream of posts. Secondly, click a hold to select the area you want to encrypt. Thirdly, right click on the selected area. Note that currently users can only use one attribute to encrypt their photos. In particular, user x can encrypt with the attribute friend(x), e.g., Alice can only encrypt with the attribute friend(Alice). Any user that is a friends with Alice has this attribute. The connection between users are as follows:

    Alice marked Bob as a friend and Charlie as family Bob marked Alice and Charlie as friends; Charlie marked Alice as friend and Bob as family.

Note that the attributes about are given according to the friend connection. Currently it is not possible to update the attributes of the users, i.e., it is not possible to create or remove friendship connections or add new users.
